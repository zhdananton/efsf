Программы питания
Для похудения

Программа похудения в 1000-1200 ккал рассчитана на получение максимальных результатов в кратчайшие сроки. Даже при минимальной калорийности рацион сбалансирован по нутриентному составу, что обеспечивает нормальную работу организма даже в условиях экстренного похудения.

Для набора массы

Идеальное соотношение белков, жиров и углеводов для качественного набора мышечной массы без лишнего жира, подходит для поддержания веса опытным спортсменам и для этапа набора начинающим атлетам.
